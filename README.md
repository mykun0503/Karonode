## インストール・起動方法
> [!IMPORTANT]
> Node.jsを導入した状態から始めてください、開発バージョンは24.14.1なのでそれ以上だったら多分動きます
> この説明はWindows用です、macOSは実機がないので無理でLinuxはUbuntu/Debian（ZorinOS）のみ書こうと思います
### 1.ダウンロードしようね
ダウンロードしましょう、多分トップ開いてると思うので一番上にあるCodeと緑色で書かれたボタンを押しましょう。クリックして多分Download ZIPがあるのでクリックしたらダウンロードされます。
### 2.ZIPなので解答しましょう（説明不要）
### 3.node.jsのモジュールのインストール
コマンドプロンプトを開いてindex.htmlがある場所に行きましょう。パスは「C:\Users\名前\Downloads\Karonode-main\Karonode-main」なはずです
```bash
npm install
npm audit fix #脆弱性があったら修正
```
上記のように実行しましょう、多分行けるはずです。
### 4.起動しよう
同じくコマンドプロンプトにて
```bash
node index.js
```
を実行すると起動できます。なんか出たら許可してください。
### 5.アクセスしよう
ブラウザで
http://localhost:3000
多分クリックしたらアクセスできます。インストールお疲れ様でした
